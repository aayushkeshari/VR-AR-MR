JPEG is used in the template, but you can switch them to `.png` by editing the image paths in `index.html` and the hero background in `style.css`.

## Publish on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files and folders from this project.
3. Go to **Settings** → **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Save.
7. Your site will be live on GitHub Pages after GitHub finishes deploying it.

## Optional edits
- Change the title or your name in `index.html`.
- Replace the hero background by changing the image in `style.css`.
- Add more screenshots by duplicating any existing `<figure>` block in `index.html`.
